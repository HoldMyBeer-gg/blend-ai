    Implement in src/blend_ai/ollama_chat.py:

    1. Add constant `SUPPORTED_IMAGE_FORMATS` as a set: `{".png", ".jpg", ".jpeg", ".webp", ".gif"}`

    2. Add function `parse_image_command(user_input: str) -> tuple[str, str] | None`:
       - If input doesn't start with "!image " (with trailing space), return None
       - Split remainder: first token is image_path, rest is message (may be empty string)
       - Return (image_path, message) or None if no path token found

    3. Add function `load_image_as_base64(image_path: str) -> str`:
       - Check extension against SUPPORTED_IMAGE_FORMATS; raise ValueError if not supported
       - Open file in binary mode, read all bytes, base64.b64encode, decode to str
       - Let FileNotFoundError propagate naturally
       - Let OSError/IOError propagate for read errors

    4. Modify `chat()` signature to accept `images: list[str] | None = None`:
       - When building the user message dict, if images is not None and non-empty, add `"images": images` to the dict
       - The message dict becomes: `{"role": "user", "content": user_message, "images": images}` when images present
       - When images is None or empty, use current behavior: `{"role": "user", "content": user_message}`

    5. In `main()` REPL loop, after the quit check and before the existing `session.chat(user_input)` call, add:
       ```
       parsed = parse_image_command(user_input)
       if parsed is not None:
           image_path, message = parsed
           if not message:
               message = "Describe this image and use it as reference."
           try:
               image_data = load_image_as_base64(image_path)
               response = session.chat(message, images=[image_data])
               print(f"\nAssistant: {response}\n")
           except FileNotFoundError:
               print(f"Error: Image file not found: {image_path}")
           except ValueError as e:
               print(f"Error: {e}")
           except OSError as e:
               print(f"Error reading image: {e}")
           continue
       ```

    Keep all existing code paths unchanged. The `base64` import already exists at line 3.
