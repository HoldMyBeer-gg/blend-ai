<objective>
Implement proactive extension suggestion system that recommends helpful free Blender extensions based on the LLM's planned task, excluding already-installed extensions.

Purpose: LLMs don't know which Blender extensions could improve their workflow. This tool enables the LLM to check what extensions would help before starting a task (e.g., suggesting Bool Tool for boolean operations, LoopTools for mesh editing) and skip suggestions for extensions the user already has installed.

Output: New handler for extension detection, new MCP tool for intelligent extension suggestions, and tests for both layers.

       Add tool function:
       ```python
       @mcp.tool()
       def suggest_extensions(task_description: str = "") -> dict[str, Any]:
           """Suggest helpful Blender extensions for a planned task.

           Analyzes the task description and recommends free Blender extensions
           that could improve the workflow. Already-installed extensions are excluded.

           Args:
               task_description: Description of the planned task. If empty, returns
                   all extensions not currently installed.

           Returns:
               Dict with 'suggestions' list of recommended extensions and
               'installed' list of already-installed extension IDs.
           """
           # Query Blender for installed extensions
           conn = get_connection()
           response = conn.send_command("get_installed_extensions")
           if response.get("status") == "error":
               raise RuntimeError(f"Blender error: {response.get('result')}")
           installed = response.get("result", {}).get("installed", [])

           # Match extensions by keyword
           task_lower = task_description.lower()
           suggestions = []
           for ext_id, info in EXTENSION_CATALOG.items():
               if ext_id in installed:
                   continue
               if not task_description:
                   # Empty task = suggest all non-installed
                   suggestions.append({
                       "extension_id": ext_id,
                       "name": info["name"],
                       "description": info["description"],
                   })
               elif any(kw in task_lower for kw in info["keywords"]):
                   suggestions.append({
                       "extension_id": ext_id,
                       "name": info["name"],
                       "description": info["description"],
                   })

           return {"suggestions": suggestions, "installed": installed}
       ```
