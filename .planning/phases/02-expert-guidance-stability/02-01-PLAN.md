       ```python
       def reset(self) -> bool:

           self._rendering.clear()
           return was_rendering
       ```

    2. In addon/__init__.py, add a load_post handler with @persistent decorator:

       - In unregister(): remove `_clear_render_guard_on_load` from `bpy.app.handlers.load_post` if present

       The @persistent decorator must be applied at module scope. Since bpy is imported inside register(), structure it as:
       ```python
       _clear_render_guard_on_load = None  # Set in register()
       ```
       Then in register():
       ```python
       from bpy.app.handlers import persistent
       global _clear_render_guard_on_load

       @persistent
       def _on_load_post(filepath):
           render_guard.on_render_complete(None)

       _clear_render_guard_on_load = _on_load_post
       bpy.app.handlers.load_post.append(_clear_render_guard_on_load)
       ```
       In unregister():

       ```python
       global _clear_render_guard_on_load
       if _clear_render_guard_on_load is not None:
           if _clear_render_guard_on_load in bpy.app.handlers.load_post:

           _clear_render_guard_on_load = None
       ```
