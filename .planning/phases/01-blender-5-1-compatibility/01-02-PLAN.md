Rewrite the Grease Pencil handler and MCP tools to use the Blender 5.x Annotation API (COMPAT-04).

Purpose: In Blender 5.x, the old unified GreasePencil type was split into GreasePencilv3 (3D objects) and Annotation (viewport overlays). Per the research decision, COMPAT-04 targets the Annotation API since the handler's functionality (layers, frames, strokes for drawing marks) matches annotation use.
Output: Rewritten gpencil handler using `bpy.data.annotations`, updated MCP tools, and tests for both layers.

       ```python
           return {"annotation": ann.name, "layer": layer.info}
       ```

       ```python
       def handle_remove_annotation_layer(params):
           """Remove a layer from an annotation."""
           ann = _get_annotation(params["annotation_name"])
           layer_name = params["layer_name"]
           layer = ann.layers.get(layer_name)
           if layer is None:
               raise ValueError(f"Layer '{layer_name}' not found on '{ann.name}'")
           ann.layers.remove(layer)
           return {"annotation": ann.name, "removed_layer": layer_name}
       ```

    6. Replace handle_add_gpencil_stroke with handle_add_annotation_stroke:
       ```python
       def handle_add_annotation_stroke(params):
           """Add a stroke to an annotation layer."""
           ann = _get_annotation(params["annotation_name"])
           layer_name = params["layer_name"]
           points_data = params["points"]
           pressure = params.get("pressure", 1.0)

           layer = ann.layers.get(layer_name)
           if layer is None:
               raise ValueError(f"Layer '{layer_name}' not found on '{ann.name}'")

           if len(layer.frames) == 0:
               frame = layer.frames.new(bpy.context.scene.frame_current)
           else:
               frame = layer.frames[0]

           stroke = frame.strokes.new()
           stroke.points.add(len(points_data))

           for i, pt in enumerate(points_data):
               stroke.points[i].co = tuple(pt)
               stroke.points[i].pressure = pressure
               # NOTE: AnnotationStroke has NO .strength property — do not set it

           return {
               "annotation": ann.name,
               "layer": layer.info,
               "point_count": len(points_data),
           }
       ```
